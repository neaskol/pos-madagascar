## ✅ Checklist de Test Rapide

### 1. Démarrage App
- [ ] App démarre sans crash
- [ ] Splash screen s'affiche
- [ ] Navigation vers onboarding/login

### 2. Authentification
- [ ] Se connecter ou compléter setup si première fois
- [ ] PIN screen fonctionne (si déjà enregistré)
- [ ] Navigation vers écran principal

### 3. Navigation vers Produits
- [ ] Accéder à l'écran Produits (via menu ou navigation)
- [ ] Liste s'affiche (vide ou avec données existantes)
- [ ] Bouton FAB "+" visible en bas à droite

### 4. Test Création Simple (Sans Photo)
- [ ] Taper sur bouton FAB "+"
- [ ] Formulaire s'ouvre
- [ ] Remplir :
  - Nom : "Test Coca-Cola"
  - Prix : 2500
- [ ] Taper "Enregistrer"
- [ ] Produit apparaît dans la liste
- [ ] Prix affiché : "2 500 Ar"

### 5. Test Création avec Photo
- [ ] Taper sur bouton FAB "+"
- [ ] Taper sur zone photo
- [ ] Permissions photo accordées (si demandé)
- [ ] Sélectionner une image dans la galerie
- [ ] Image apparaît dans le formulaire
- [ ] ⏱️ Attendre upload (indicateur de chargement)
- [ ] Remplir :
  - Nom : "Test Pizza"
  - Prix : 15000
- [ ] Taper "Enregistrer"
- [ ] Produit avec photo apparaît dans la liste

### 6. Test Édition
- [ ] Taper sur un produit existant
- [ ] Formulaire d'édition s'ouvre avec données
- [ ] Modifier le nom ou le prix
- [ ] Taper "Enregistrer"
- [ ] Retour à la liste
- [ ] Modifications visibles dans la liste

### 7. Test Recherche/Filtres
- [ ] Taper dans la barre de recherche
- [ ] Taper quelques lettres
- [ ] Liste filtrée en temps réel
- [ ] Effacer recherche
- [ ] Liste complète réapparaît

### 8. Test Indicateurs Stock (si produits avec stock)
- [ ] Vérifier couleurs des badges stock :
  - 🟢 Vert : stock normal
  - 🟠 Orange : stock bas
  - 🔴 Rouge : rupture
- [ ] Quantités affichées correctement
